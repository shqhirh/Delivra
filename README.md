Delivra — Parcel Management System

A. Project Title & Description

Project Title: Delivra — Parcel Management System

Description:
Delivra is a web-based parcel management system designed to manage parcel tracking, customer information, and user account settings in a single dashboard interface. The system provides a clean and responsive user interface for easy navigation and data interaction.

This project is developed using frontend web technologies and is intended for educational purposes to demonstrate dashboard UI design, user interaction, and data management using browser storage.

B. Features Included
1. Login Page (index.html)
- Admin login interface
- Simple authentication (demo purpose)
- Redirects to dashboard upon successful login
- Responsive and user-friendly design
  
2. Dashboard (dashboard.html)
- Main control panel after login
- Sidebar navigation menu
- Notification system
- User profile dropdown
- Quick access to all system modules
  
3. Parcels Tracking Page (parcels.html)
- Parcel list display
- Search and filter functionality
- Parcel status tracking view
- Organized layout for easy viewing
  
4. Customers Page (customers.html)
- Customer information displayed in card format
- Search functionality by name, email, location, or order ID
- Responsive grid layout
- Displays customer delivery statistics
  
5. Settings Page (settings.html)
- Update user profile (name and email)
- Manage saved addresses (add, edit, delete, set default)
- Toggle settings:
- Dark mode
- Email notifications
- Parcel updates
- Reset preferences option
- Logout function
- Data stored using browser localStorage

Summary

Delivra is a simple dashboard system that helps manage parcels, customers, and settings. It is a frontend project used for learning how web applications work.
